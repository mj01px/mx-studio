## `~/about`

```ts
const mxStudio = {
  type:     "Landing Page",
  stack:    ["React", "TypeScript", "Vite", "Tailwind CSS", "Framer Motion"],
  features: ["Scroll-linked video scrubbing", "Bento grid", "Glassmorphism", "AI-generated assets"],
  branding: "Logitech MX Series · Dark Navy #101b4f",
  author:   "Mauro Junior · github.com/mj01px",
} as const;
```

**MX Studio** is a high-performance landing page built for the Logitech MX series ecosystem. It features a cinematic scroll-linked video experience driven by Framer Motion, a premium bento grid layout, and liquid glassmorphism components — all designed around a dark navy aesthetic.

## `~/getting-started`

```bash
git clone https://github.com/mj01px/mx-studio.git
cd mx-studio
npm install && npm run dev     # → http://localhost:5173
```
